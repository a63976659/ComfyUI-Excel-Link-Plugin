## 🚀 ComfyUI-Excel 联动插件：自动化工作流的核心枢纽

**Excel 联动插件** 是一款专为提升 ComfyUI 生产力而设计的工具。它打破了传统 AI 生成中“手动调整参数”的瓶颈，让你可以通过一份 Excel 表格，实现**自动化批量生成、数据追踪、以及素材的精准管理**。

### 🛠️ 核心功能一览

* **📊 数据深度联动**：直接从 Excel 中读取提示词、控制参数或行号，并将生成结果（路径、时间、参数）反向写回表格，实现生产流程闭环。
* **🖼️ 图像精准插入**：支持将生成的图像自动插入到 Excel 指定单元格。具备独特的“匹配单元格”模式，让图片自动适应表格宽高，确保排版整齐且不失真。
* **⏳ 自动化时间管理**：提供秒级精度的系统时间获取与 Excel 专用格式转换，方便记录每一张图的准确生成耗时。
* **🧩 全能字符串/运算工具**：集成了精简高效的 19 种字符串处理节点（如提取参数、批量替换、过滤行）及数值规范化工具，轻松应对复杂的逻辑判断需求。
* **🎭 场景化增强**：内置文字图像生成（字幕制作）、完成音频提醒等功能，极大地优化了长时间挂机生成的使用体验。

### 📥 安装方法 (Installation)

#### 方法一：Git Clone（推荐）

1. 进入 ComfyUI 自定义节点目录：
```bash
cd ComfyUI/custom_nodes/

```


2. 克隆仓库：
```bash
git clone https://github.com/a63976659/ComfyUI-Any-Path-Repair.git

```


3. 重启 ComfyUI。

#### 方法二：手动下载

1. 点击 GitHub 页面上的 **Code → Download ZIP**。
2. 解压到 `ComfyUI/custom_nodes/` 目录。
3. 重启 ComfyUI。

### 💡 快速上手场景

1. **批量动画制作**：在 Excel 中编写每一帧的场景描述，利用插件循环读取并连续生成故事板。
2. **模型/参数测试**：自动记录每个生成参数对应的图片路径，方便后期对比筛选。
3. **电商素材出图**：根据 Excel 中的产品清单，批量处理文件名并叠加 Logo 装饰层。
